* Goal: Jira-like BidOps platform for ITSQ to manage the full bid lifecycle from sourcing and purchase to submission and outcome, with SLA timers, approvals, compliance, pricing, and a document vault. Enrich with external award results from public portals to build competitor intelligence. Host in Azure Qatar Central.
* Scope: Web app, background workers, file processing, award-result collectors, Power BI analytics, SSO, and audit. Import current tracker to day-one populate.
* Success: No missed deadlines. Consistent submission packs. Clean audit of approvals. External awards ingested with low parsing error. Executive dashboards reflect live KPIs.
  * 100% on-time submissions during UAT and after go-live.
  * ≥ 98% field mapping accuracy when importing the current tracker.
  * ≥ 200 award records ingested for first portal with ≤ 2% numeric parse error after curation.
  * Power BI dashboards refresh automatically and align with DB snapshots.
* Out of scope v1: ERP functions, invoicing, procurement payments, post-award project execution beyond closeout.
